**笔记**

1. 只要打开云外壳终端，服务器就可以工作（在rdp上安装chrome并打开那里的外壳终端进行修复）。
2. 服务器每周只能工作50个小时，因此，如果您想始终使用它，请使用2-3个Google帐户。
3. 如果重新启动服务器，则需要再次执行所有步骤。
4. 请与我联系以获得帮助。
